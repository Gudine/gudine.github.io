Habilidades

Python

Python foi a primeira linguagem que eu aprendi, e já tenho uma grande familiaridade com ela. Sei muitos dos métodos e funções de cór, e os que não sei, consigo achar com algumas pesquisas rápidas.

JavaScript

No JavaScript, em comparação ao Python, sou um pouco mais amador, mas as aulas da Trybe com certeza me farão ser tão bom nele quanto em Python, ou até melhor. A sua variedade de usos também significa que eu não poderia perder esse 'canivete' das linguagens.

HTML

HTML também é um conhecimento recente para mim, mas eu já conheço bastante dele. Sou capaz de criar páginas básicas e médias, fazendo uso de tags semânticas para facilitar a leitura por métodos alternativos.

CSS

Mesmo que recente, acredito que meu conhecimento sobre CSS seja bem alto. Estou acostumado a usar Grid e Flex, e sei de muitos modos de conseguir uma aparência desejada. No entanto, meu senso de estética não é tão bom...

Projetos

Clique na imagem para acessar a página do projeto.

Exercícios da Trybe

Esses exercícios, feitos no decorrer do curso da Trybe, serviram para praticar as várias habilidades que foram adquiridas por mim. Inclui muitos exemplos de HTML, CSS, JavaScript, e irá incluir muito mais com o tempo.

Carrinho de Compras

Esse é um projeto que simula uma página de compras, carregando informações de produtos de forma assíncrona através de uma API. Foi feito como uma espécie de 'desafio final' do módulo de Fundamentos da Trybe, envolvendo, além do assincronismo, uso de HOFs e conhecimento de HTML e DOM.

Zoo Functions

Esse é um projeto no qual é possível acessar informações sobre o funcionamento de um zoológico. Foi feito para praticar o uso efetivo de informações de objetos, além do uso de HOFs (Funções de Ordem Elevada).

Testes Unitários em JavaScript

Esse é um projeto feito para praticar meus conhecimentos de Jest, alternadamente criando funções para serem testadas e testes das funções. Também pratiquei o uso do TDD (Desenvolvimento Guiado por Testes).

Trybewarts

Essa é uma página baseada na fantástica escola de magia Trybewarts, uma paródia da franquia Harry Potter criada por J.K. Rowling. A página contém um grande formulário e possui algumas funções de JavaScript de cunho mais avançado.

Carta Misteriosa

Essa é uma página feita para simular uma carta misteriosa, que comummente é feita a partir de recortes de jornais e revistas. O usuário insere um texto na página, e a página exibe uma carta misteriosa com esse texto.

Adivinhe a Cor

Esse é um pequeno jogo no qual uma cor é dada no formato rgb e o jogador deve clicar no círculo que contém a cor correta, ganhando ou perdendo pontos conforme vai jogando.

Gerador de Memes

Essa é uma página feita para permitir que o usuário crie memes. É possível escolher uma imagem (dentre algumas pré-selecionadas ou fazendo upload de uma) e digitar um texto que aparece em cima, além de escolher bordas para a imagem.

Lista de Tarefas

Essa é uma página feita para disponibilizar ao usuário um lugar para anotar suas tarefas, com recursos simples como editar tarefas, mudar a ordem delas e marcá-las como finalizadas.

Arte com Pixels

O primeiro projeto que integra o HTML/CSS com JavaScript, com o intuito de possibilitar que o usuário faça desenhos simples em pixels, com uma pequena variedade de cores e algumas configurações disponíveis.

Playground Functions

Esse foi um projeto feito exclusivamente com JavaScript, pensado para testar as minhas habilidades no mesmo, fazendo várias funções simples que seguem as especificações dadas. A página web que interpreta os comandos foi feita posteriormente.

Lições Aprendidas

Esse foi o meu primeiro projeto feito na Trybe, e exibe minhas habilidades básicas de HTML e CSS, sem fazer uso de opções mais avançadas como Flex/Grid, ou HTML semântico.
